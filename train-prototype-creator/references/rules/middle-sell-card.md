# Middle Sell-Card Rules

本文件约束中间页 `sell-card-*` 预订方案卡。来源：Figma `KAzZgNZsvzGWmAB67Qflgm` / node `214:6636`，命名包含 `sell-card` 的内容。

## 结构

默认结构为三列：

```text
左侧价格区 → 中部权益区 → 右侧订按钮
```

- 卡片列表左右页边距：8px。
- 卡片间距：8px。
- 普通卡高度：72px。
- 高权益卡高度：118px。
- 省钱对比卡高度：96px，底部可带 30px 浅橙权益条。

## 色彩

- 业务蓝：`#006FF6`，用于价格、座席选中态。
- 预订按钮蓝：`#3263A6`，用于所有右侧“订 / 预订”按钮。
- 主橙：`#FF7700`，用于抢票、权益强调、`享` 图标。
- 补贴橙红：`#FF5F26`，用于补贴金额。
- 正文黑：`#111111`。
- 次级灰：`#555555`。
- 弱提示灰：`#888888` / `#AAAAAA`。
- 分隔与弱点：`#DCE4ED`。
- 页面背景：`#F5F5F5`。
- 卡片背景：`#FFFFFF`。
- 浅橙背景：`#FFFAF5` / `#FFF2E9`。

## 字体

- 中文 UI：`PingFang SC`。
- 数字价格与时刻：`tripnumber` 优先，回退到 `PingFang SC`。
- 价格数字：24px / 30px，半粗。
- 价格货币符号：16px / 16px，半粗。
- 权益正文：13px / 17px。
- 补贴 Tag：12px / 15px，半粗。
- 按钮文字“订”：19px，半粗。

## 间距

- 卡片内边距：普通卡 `10px 16px`。
- 三列间距：左价区与权益区 16px，权益区与按钮 16px。
- 权益行间距：6px。
- 权益图标与文案间距：6px。
- 价格与补贴 / 附加说明的垂直间距：2px。
- 底部权益条左右间距：16px。

## 圆角与组件风格

- sell-card 外层圆角：8px。
- 右侧“订”按钮：44px × 44px，圆角 8px，`#3263A6` 蓝底白字。
- 补贴 Tag：2px 圆角、浅橙底、橙红描边。
- `@svg-middle-sellcard-xiang`：16px × 16px，4px 圆角，橙底白字。
- 底部省钱权益条：浅橙底，仅随卡片底角继承圆角。
- 权益列表使用 4px 浅蓝灰圆点；会员权益可使用绿色圆形图标。

## 交互状态

- 默认态：白底卡片 + 蓝色价格 + 蓝色订按钮。
- 按压态：按钮或整卡缩放至 `0.985`，不改变信息布局。
- 选中 / 推荐态：优先通过卡片排序、补贴 Tag 和权益强调表达，不额外增加大面积描边。
- 不可售 / 弱化态：价格和按钮降为灰色，权益保留但弱化到 `#888888`，不得隐藏价格。
- 展开态：末行可显示“共 N 项”与展开箭头，展开箭头必须走 SVG 资产。

## SVG 资产约束

以下 Figma `@svg` 编组必须以 SVG 资产调用：

- `@svg-middel- selling card-vs` → `middle-selling-card-vs.svg`
- `@svg-middle-sellcard-xiang` → `middle-sellcard-xiang.svg`
- `@svg-icon-问号` → `middle-icon-help.svg` 或公共 `icon-q&a.svg`

不得使用 emoji、裸字符、CSS 三角或手绘路径替代上述资产。
